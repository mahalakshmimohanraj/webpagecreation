About This Project

This project demonstrates how to work with text in HTML. You can use paragraphs, headings, lists, and more.

  • Paragraphs: <p>...</p>
  • Headings: <h1> to <h6>
  • Lists: <ul>, <ol>, <li>
  • Bold: <strong> or <b>
  • Italic: <em> or <i>
